# Hello World App for Redmi Note 7

A modern, production-ready Android application optimized for Redmi Note 7 and devices running Android 7.0 (API 24) and above.

## 📱 Features

- **Modern UI** with Material Design 3 components
- **MVVM Architecture** with LiveData and Coroutines
- **Smooth Animations** for enhanced user experience
- **Error Handling** with user-friendly messages
- **Empty State UI** for better UX
- **Responsive Layout** optimized for Redmi Note 7 (1080x2340)
- **Adaptive Launcher Icons** for Android 8.0+
- **Professional Error Handling** with proper permissions

## 🛠️ Tech Stack

- **Language**: Kotlin
- **Minimum SDK**: 24 (Android 7.0)
- **Target SDK**: 34 (Android 14)
- **Architecture**: MVVM
- **UI**: XML Layouts with Material Design Components
- **Dependencies**: AndroidX, Coroutines, LiveData, ViewModel, Glide

## 📦 Installation

### From Source

```bash
# Clone the repository
git clone https://github.com/yourusername/HelloWorldApp.git

# Open in Android Studio
cd HelloWorldApp

# Build debug APK
./gradlew assembleDebug

# Build release APK
./gradlew assembleRelease
